Why inline, not modal

Modals steal focus, break the conversation thread, and force you to re-read context after dismiss. Decision chips live where the request was made — answer is one keystroke, the rest of the surface stays exactly as it was.

04

Discipline · what we deliberately don't do

The 2026 cohort is converging on patterns that benchmark well in launch demos and corrode daily flow.
Anti-pattern · 1

No separate Mission Control view.

Antigravity and Cursor 3 split agent state into a second dashboard. Atrium keeps agent status on the pipeline tile. One look at the Overview tells you which wave is alive, with a green pulse on the active card — no second tab to babysit.

avoided · Antigravity · Cursor 3 Glass
Anti-pattern · 2

No effort-mode dropdowns as chrome.

Replit and Cursor expose Lite / Economy / Power as a top-level control on every prompt. Atrium has one control — the Mode pill (default · acceptEdits · plan · bypass). Model is set once per project; you don't pay a decision tax on every send.

avoided · Replit Agent 3 · Cursor
Anti-pattern · 3

No cloud-only background agents as headline.

Codex web and Cursor's background agents pull the mental model toward "PR notifications later." Atrium is local-first, synchronous, trusted. Worktrees give us isolation without delegation. The agent is in the room with you, not in some queue.

avoided · Codex web · Cursor background
Contrarian · the big one

One agent, one canvas, one conversation.

Everyone is converging on parallel-agent grids. Reviews say the same thing — context-switching cost dominates, merges get gnarly, you become a load balancer. Atrium's two-tier surface implies serial focus. We lean in. The pipeline shows progress across stages; you only ever look one agent in the eye at a time. Taste means saying no to a feature that benchmarks well and corrodes daily flow.
