First-class drawer.

MCP marketplace

2026-05-04· 6 tickets· Owner · you

Why now

Atrium currently inherits the user's installed MCP servers. To be product-friendly, we want a discoverable marketplace inside the app — installs in two clicks, surfaces capability before commitment.

Acceptance criteria

  • ✓List installed MCP servers and their tools, grouped by server.STO-2138
  • ✓Show tool count, last activity, and "in use by N skills" for each server.STO-2139
  • Browse a curated registry of community MCPs — read-only catalogue, no auto-install.STO-2140
  • One-tap install flow that writes to the user's ~/.claude/mcp.json with explicit confirmation.STO-2141
  • Per-server permission preview · which Bash, Edit, and Read paths the MCP can touch.STO-2142
  • Telemetry off by default; explicit toggle for usage metrics.STO-2143

Out of scope

Publishing flow for community MCPs · server-side ratings or reviews · payment-gated MCPs.

Animation reference

Six motions across the whole product. Cap concurrent motion to one element on screen.

Six motions, six purposes.

Every animation in Atrium answers a state question — where Claude is, what's selected, where a wave just went. If a motion doesn't answer one of those, it doesn't ship.

M-01
Breathing caret

The "‹" inside the active todo's tick. Says: Claude is mid-sentence here. 1.8s ease-in-out, opacity 0.45→1.0.

›
M-02
Live pulse

The green dot on the live-chip. Says: this wave has Claude attached. 2.4s ease-in-out box-shadow ramp.

M-03
Cursor block

The block at end of the active todo's sub-line — and inside the terminal. 1.05s blink. Speaks the same dialect as the terminal.

›
M-04
Cell breathe

Background of the current journey-ribbon cell. 4.2s slow breathe. Tells you which stage you're in without competing with the canvas.

M-05
Pin glow (once)

When a pin is focused. Plays once on focus, then sits still — no infinite ring. 700ms ease-out-expo.

M-06
Loop-back glide

The wave card sliding stage→stage. 480ms cubic-bezier(0.32, 0.72, 0.24, 1). Brief opacity dip mid-flight.
